# 📖 Medibox 智能药箱 - 用户使用手册

> **写给用户的话**：
> 感谢您选择 Medibox！本手册将帮助您快速上手使用智能药箱。我们将尽量使用简单易懂的语言，如有任何疑问，请随时联系我们的支持团队。

## 🎁 第一步：开箱与安装

### 1. 检查包装清单
打开包装盒，请确认您收到了以下物品：
- [ ] **1 个 主控中心 (Hub)**：带有电源插孔的大盒子。
- [ ] **若干个 智能药盒 (Modules)**：您购买的小药盒（通常为 3-5 个）。
- [ ] **1 个 电源适配器**：用于连接插座。
- [ ] **说明书**：即本文档。

### 2. 组装药箱
Medibox 采用磁吸设计，安装非常简单：
1. 将 **主控中心** 放在桌面上。
2. 拿起一个 **智能药盒**，将其侧面的金属触点对准主控中心的侧面触点。
3. 轻轻靠拢，它们会自动吸附在一起。
4. 您可以继续将其他药盒依次吸附连接，组成长条形或方阵形。

### 3. 接通电源
1. 将电源适配器插入家里的电源插座。
2. 将另一端插入主控中心的电源孔。
3. 观察 **主控中心** 上的指示灯，它应该开始闪烁（表示正在启动）。

---

## 🌐 第二步：连接网络 (仅需一次)

为了让药箱能通过手机控制，需要将其连入家里的 WiFi。

1. **连接热点**：打开您的手机 WiFi 设置，找到名为 `Medibox-Setup` 的网络并连接（无需密码）。
2. **填写信息**：
   - 连接成功后，手机会自动弹出一个网页（如果没有，请打开浏览器访问 `192.168.4.1`）。
   - 在网页上选择您家里的 WiFi 名称。
   - 输入 WiFi 密码。
   - 点击“保存并连接”。
3. **验证成功**：观察主控中心的指示灯，如果变成 **常亮绿色**，说明网络连接成功！

---

## 🗺️ 第三步：设置药盒 (Blink to Identify)

现在，我们要告诉手机，哪个盒子装了什么药。这个过程叫“映射”。

**准备工作**：确保手机和药箱连在同一个 WiFi 下。

1. **打开控制台**：在手机浏览器输入 `http://medibox.local`。
2. **添加药盒**：点击屏幕上的 **“添加药盒”** (Add Pillbox) 按钮。
3. **观察闪烁**：此时，看一眼您的实物药箱，会有一个药盒的灯正在 **闪烁蓝光**。
4. **点击对应位置**：在手机屏幕的网格上，点击那个正在闪烁蓝灯的对应位置。
5. **输入信息**：手机会提示输入药品名称（例如“降压药”），输入后点击确认。

*重复以上步骤，直到所有药盒都添加完毕。*

---

## ⏰ 第四步：设置提醒

1. 在手机界面上，点击刚才添加的 **“降压药”** 卡片。
2. 点击 **“设置提醒”** (Set Reminder)。
3. 选择时间（例如 `08:00`）。
4. 输入用量（例如 `2 粒`）。
5. 点击 **“保存”**。

---

## 💊 日常使用指南

当到了吃药时间（例如早上 8 点）：

1. **听到语音**：药箱会大声说：“**该吃药了，请服用降压药。**”
2. **看到灯光**：装有降压药的那个盒子，灯光会 **闪烁白光**。
3. **打开药盒**：
   - 打开那个闪烁白光的盖子。
   - 药箱会再次语音提示：“**请服用 2 粒。**”
4. **服药**：取出药物服用。
5. **关闭药盒**：关上盖子，灯光熄灭，提醒结束。

## ❓ 常见问题 (FAQ)

**Q: 药盒的灯不亮了怎么办？**
A: 请检查电源插头是否插紧，或者重新插拔一下药盒模块。

**Q: 提醒时间到了，但是没有声音？**
A: 请检查主控中心的音量设置，或者确认是不是停电了。

**Q: 换了新的 WiFi 怎么办？**
A: 长按主控中心背后的“Reset”按钮 5 秒钟，直到灯光变回闪烁状态，然后重新进行“第二步：连接网络”。

**Q: 我想带药盒出门，需要一直插电吗？**
A: 本设备设计为家用固定使用，建议一直连接电源。如果短暂断电，时间会自动保存。

## 🔧 简单故障排查

如果遇到问题，请按以下流程尝试：

1. **断电重启**：拔掉电源，等待 10 秒，再插上。
2. **检查连接**：把所有药盒拆开，重新吸附连接一遍。
3. **重置网络**：如果手机连不上药箱，请长按 Reset 键重置网络。
